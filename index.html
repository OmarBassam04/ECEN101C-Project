Ohm's Law!

Imagine a highway as an electrical circuit. The number of cars traveling on the highway represents the current, the speed limit of vehicles represents the voltage, and the number of lanes on the road represents the resistance. Let's explore this analogy further!






Current: 0
Resistance: 0
Voltage: 0

Hence, the average speed limit is 0 kph.

The current is 0 Amperes (cars).

The resistance is 0 Ohms (lanes).

The voltage is 0 Volts (speed).


With Ohm's Law calculation : Voltage (V) = Current (I) x Resistance (R)
= 0 x 0 = 0 volts.
